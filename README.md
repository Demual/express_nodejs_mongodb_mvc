# Crear Servidor y conectar a MongoDB

## Descripción
* Programa para hacer una conexión al servidor y a la base de datos MongoDB, usando Express, Mongoose y NodeJS, modelo MVC

## Tecnologías
* NodeJS
* Express
* Mongoose
* Compression
* Nodemon
* Cors
* Helmet

## Ejecución
* Importar módulos con npm install
* Usar node index para ejecución simple
* Usar npm start, para ejecución del programa
* Usar npm run dev, para ejecutar con nodemon
* Pobrar el Server en http://localhost:3000/
